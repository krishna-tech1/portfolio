# Professional Portfolio

A minimal, high-performance portfolio website built with **Python (Flask)**, **HTML5**, and **Vanilla CSS**.

## Features
- **Responsive Design**: Looks great on mobile and desktop.
- **Glassmorphism**: Modern UI design trends.
- **Smooth Scrolling**: Enhanced user experience.
- **Easy Customization**: Edit the HTML to add your content.

## Setup Instructions

### 1. Prerequisites
Python is required. (It has been installed on your system).

**IMPORTANT**: If you just installed Python, **restart VS Code** or your terminal for it to be recognized.

### 2. Install Dependencies
Open your terminal in this folder and run:
```bash
pip install -r requirements.txt
```

### 3. Run the Application
Start the local server:
```bash
python app.py
```

### 4. View Portfolio
Open your browser and navigate to:
[http://127.0.0.1:5000](http://127.0.0.1:5000)

## Customization
- **Content**: Edit `templates/index.html`.
- **Styles**: Edit `static/css/style.css`.
- **Logic**: Edit `app.py` for backend features.
